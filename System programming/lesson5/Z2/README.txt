Это 2 задание 5 урока

Задание: На вход программе подаётся путь к файлу. В файле будут 10 строк из случайных 10значных чисел. Главная программа открывает файл и создаёт 3 потока, которые будут считывать строку и разворачивать её. (Заблокировал доступ к файлу, считал строку, открыл доступ к файлу, развернул строку, вывел развёрнутую строку и свой tid). Программа работает пока не выведет все строки из файла в развёрнутом виде.

Пример выполнения программы:

~$ make

~$ ./run numbers.txt

~$ make clean (Удаление исполняемых файлов)
