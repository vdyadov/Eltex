Задание: На вход программе подаётся строка с несколькими словами. Главный процесс создаёт столько потоков, сколько слов в строке и передаёт им эти слова в аргументах. Потоки должны развернуть эти слова в обратном порядке. (Как в первом задании, только на потоках)

Пример выполнения программы:

~$ make

~$ ./run Hello it works

~$ make clean (Удаление исполняемых файлов)
