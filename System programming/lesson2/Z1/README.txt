Это 1 задание 2 урока

Задание 1: Процесс родитель создает матрицу 5х5, заполняет случайными числами, создаёт процессам потомкам через канал, ждёт ответа от потомков и выводит его. Потомки: 1) Поиск самого большого числа. 2) самого маленького числа 3) Сумма всех чисел 4) произведение всех чисел.

Пример выполнения программы:

~$ make

~$ ./run

~$ make clean (Удаление исполняемых файлов)
