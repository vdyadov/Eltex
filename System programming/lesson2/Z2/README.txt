Задание 2: Написать чат для двух программ: Открываются 2 терминала запускают программу чат со входными параметрами. Что пишется в одном терминале, то выводится и в другом.

Пример выполнения программы:

~$ make

~$ ./run -r (В том терминале, куда будем вписывать текст)

~$ ./run -w (В том терминале, где будет выводиться этот текст)

~$ make clean (Удаление исполняемых файлов)
