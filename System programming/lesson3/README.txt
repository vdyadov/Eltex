Это задание 3 урока

Задание: Используя очереди сообщений создать сервер, который будет постоянно слушать одну очередь сообщений. В эту очередь будут поступать сообщения с новым key и именем поступившего пользователя. Сервер при поступлении нового пользователя начинает слушать key который пришёл в сообщении на регистрацию. При поступлении сообщения от конкретного клиента сервер дублирует это сообщение всем остальным подключённым клиентам.

Пример выполнения программы:

~$ make

~$ ./run -r (В том терминале, куда будем вписывать текст)

~$ ./run -w (В том терминале, где будет выводиться этот текст)

~$ make clean (Удаление исполняемых файлов)
