Это 2 задание 4 урока

Задание: Создать программу в которой процесс родитель создаёт разделяемую память и записывает туда 0. Также процесс родитель создаёт столько потомков, сколько файлов в директории (где запустился процесс). Процессы потомки читают файл с текстом. При каждом прочтении символа процесс потомок обращается к разделяемой памяти, извлекает число, которое там записано и увеличивает его на 1. В итоге после завершения всех потомков процесс родитель выводит на экран число из разделяемой памяти.

Пример выполнения программы:

~$ make

~$ ./run 

~$ make clean (Удаление исполняемых файлов)
