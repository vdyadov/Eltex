Это 1 задание 3 урока

Задание: Процесс родитель создает матрицу 5х5, заполняет случайными числами, записывает в разделяемую память, создаёт потомков: 1) Поиск самого большого числа. 2) Самого маленького числа 3) Сумма всех чисел 4) Произведение всех чисел. Весь код потомков должен быть записан в другом файле и вызван через exec. Ключ памяти и код выполняемой задачи передаются через входные аргументы. Процесс родитель заканчивает работу после того как все потомки закончили работу.

Пример выполнения программы:

~$ make

~$ ./run 

~$ make clean (Удаление исполняемых файлов)
