Задание 3: Создать программу, которую будут выполнять потомки. В зависимости от входных аргументов программа должна выполнять: 1) Вывод матрицы заполненную нулями(размер передаётся в аргументах к вызову) 2)Вывод строки (так же в аргументах) 3) Отсортировать массив (так же в аргументах).

Пример выполнения программы:

~$ make build (собирает все файлы)

~$ make test1 (Запускает 1 вариант программы)

~$ make test2 (Запускает 2 вариант программы)

~$ make test3 (Запускает 3 вариант программы)

~$ make clean (Удаление исполняемых файлов)
