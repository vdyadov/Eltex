4: Пары замен символов считываются из файла, имя которого является дополнительным параметром и могут выглядеть так: арбквъ … 4759. Регистр первого символа слова сохраняется.

1: Знаки препинания заменяются согласно последовательности, являющейся дополнительным параметром программы, например: !*,.?;)(+–:"/' (здесь символ «!» заменяется на символ «*», «*» – на «,», … «'» – на «!»

4: Из командной строки, недостающие параметры вводятся с консоли

Пример запуска: ./run -z '][}{)(*&^%$#@!?' -i input.txt -o output.txt -f code.txt

input.txt - входной файл.
output.txt - выходной файл.
code.txt - файл с парами букв и чисел.

Для дешифроки используется же этот код, но только на вход подается вывод предыдущей программы.

Пример запуска: ./run -z '][}{)(*&^%$#@!?' -i output.txt -o output1.txt -f code.txt
